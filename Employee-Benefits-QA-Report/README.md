Paylocity Benefits Dashboard – QA Bug Report

This repository contains the complete QA analysis, including UI and API bug reports, observations, and supporting screenshots for the Paylocity Benefits Dashboard.

Contents
- **UI_Bug_Report.md** – All UI defects with reproduction steps and screenshots  
- **API_Bug_Report.md** – All API defects with reproduction steps and screenshots  
- **/screenshots** – All referenced screenshots  

Summary
The purpose of this testing effort was to validate:
- Add, Edit and Delete operations via UI and API  
- Data consistency between UI and API  
- Validation rules  
- Error handling  
- Business rule alignment  

All findings are documented with severity, labels, expected vs actual results, and recommendations.
